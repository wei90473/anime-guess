# anime-guess

## 產品定位

動漫猜謎小遊戲。玩家選擇作品後，進行混合題型猜謎（選擇題 + 填字）。

## 核心流程

```
選擇作品 → 混合題型猜謎（角色/劇情/台詞）→ 看結果
```

## 目標用戶

動漫愛好者，訪客可直接玩，無需帳號。

## 功能範圍

### MVP 必做
- 作品目錄（管理員維護）
- 進入作品開始猜謎
- 選擇題題型
- 填字題型
- 玩家投稿題目
- AI 規則初篩 + 管理員人工審核

### 不做
- 分數 / 排行榜
- 帳號系統（訪客直接玩）
- 串接外部 API
- 付費功能

## 技術方向

- 後端：Native PHP + MySQL
- 前端：Server-rendered PHP + Vanilla JS
- 手機優先設計
